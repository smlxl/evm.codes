## Stack input

0. `Integer`: first value to add.
1. `Integer`: second value to add.

## Stack output

0. `Integer`: result of the addition modulo 2<sup>256</sup>.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| 10 | 20 | * | 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF | 0 |
| 10 | | * | 1 | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
