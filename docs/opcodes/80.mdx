## Stack input

0. `value`: value to duplicate.

## Stack output

0. `value`: duplicated value.
1. `value`: original value.

## Example

| Input | Output |
|------:|-------:|
| `1` | `1` |
| | `1` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
- Stack overflow.
