*Index 1 is top of the stack. See [PUSH](/#60).*

## Stack input

0. `b`: size in byte - 1 of the integer to be sign-extended.
1. `x`: integer value to sign extend.

## Stack output

0. `y`: integer result of the sign extend.

## Examples

| * | Input | Output | * | * | Input | Output |
|--:|------:|-------:|--:|--:|------:|-------:|
| `1` | `0` | `0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF` | * | `1` | `0` | `0x7F` |
| `2` | `0xFF` | | * | `2` | `0x7F` | |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
