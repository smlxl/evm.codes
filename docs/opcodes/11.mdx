## Stack input

0. `Integer`: left side value.
1. `Integer`: right side value.

## Stack output

0. `Boolean`: 1 if the left side is bigger, 0 otherwise.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| 10 | 1 | * | 10 | 0 |
| 9 | | * | 10 | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
