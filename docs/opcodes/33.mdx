## Stack output

0. `Integer`: the 20-byte address of the caller account. This is the account that did the last call (except [delegate call](/#F4)).

## Example

| Input | Output |
|------:|-------:|
|  | 0x5B38Da6a701c568545dCfcB03FcB875f56beddC4 |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Stack overflow.
