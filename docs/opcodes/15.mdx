## Stack input

0. `Boolean`: input.

## Stack output

0. `Boolean`: 1 if the input is 0, 0 otherwise.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| 10 | 0 | * | 0 | 1 |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
