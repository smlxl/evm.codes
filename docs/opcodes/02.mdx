## Stack input

0. `Integer`: first value to multiply.
1. `Integer`: second value to multiply.

## Stack output

0. `Integer`: result of the multiplication modulo 2<sup>256</sup>.

## Examples

| Input | Output |
|------:|-------:|
| 10 | 100 |
| 10 | |

| Input | Output |
|------:|-------:|
| 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFF | 0xFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFFE |
| | 2 | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
