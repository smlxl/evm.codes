## Notes

All values are treated as two's complement 256-bit signed integers. Note the overflow semantic when âˆ’2<sup>255</sup> is negated.

## Stack input

0. `a`: integer numerator.
1. `b`: integer denominator.

## Stack output

0. `a % b`: integer result of the signed integer modulo. If the denominator is 0, the result will be 0.

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
