## Stack input

0. `a`: value to swap.
1. `b`: ignored value.
2. `c`: value to swap.

## Stack output

0. `c`: swapped value.
1. `b`: ignored value.
2. `a`: swapped value.

## Example

| Input | Output |
|------:|-------:|
| `1` | `2` |
| `0` | `0` |
| `2` | `1` |

## Error cases

The state changes done by the current context are [reverted](#FD) in those cases:
- Not enough gas.
- Not enough values on the stack.
