## Stack input

0. `Integer`: first value.
1. `Integer`: second value.

## Stack output

0. `Integer`: the bitwise AND result.

## Examples

| Input | Output | * | Input | Output |
|------:|-------:|-|------:|-------:|
| 0xF | 0xF | * | 0xFF | 0 |
| 0xF | | * | 0 | |

## Error cases

The transaction is reverted in those cases:
- Not enough gas.
- Not enough values on the stack.
